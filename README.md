# Digital Memory Architecture: RAM Decoder Simulation

This project features a comprehensive simulation of a **Digital Memory Architecture**, specifically focusing on the design and implementation of a **RAM Decoder** system. Developed using **Matlab & Simulink**, the model demonstrates the fundamental principles of digital logic and memory addressing.

---

### 🔍 Project Overview

In this simulation, I designed a functional RAM architecture that processes addressing logic to store and retrieve data efficiently. The project highlights the intersection of computer engineering principles and practical hardware simulation.

**Key Technical Features:**
- **Memory Decoding:** Implementation of complex logic gates to manage memory address selection.
- **System Simulation:** Modeled entirely within **Matlab/Simulink** to ensure logical accuracy and timing precision.
- **Visual Verification:** Integrated display outputs (as seen in the schematic) to verify data flow and memory states in real-time.
